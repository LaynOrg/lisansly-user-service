## Lisansly User API

### Running Locally
Before run app you must set environment variables in .env file then you can running app just like this:

```shell
make run
# or
make build
```

### Linting
Needs golangci-lint package installed locally

```shell
make lint
```

### Testing

```shell
make test
```

### Git Hooks:
Needs pre-commit package installed locally

Installation:
```shell
pre-commit install
```

Run:
```shell
pre-commit run
```
